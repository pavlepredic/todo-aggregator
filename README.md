# Todo aggregator

# Tech stack

### Backend

- Symfony 4

### Infrastracture

- Kubernetes
- MariaDB
- nginx

# Development

- Clone
- Run `make up`
- Access app at [http://localhost/](http://localhost/)
- Run tests using `./infrastructure/dev/run phpunit`
- Execute symfony console commands using `./infrastructure/dev/run console`
- Add composer dependencies using `./infrastructure/dev/run composer`
- Run migrations using `./infrastructure/dev/run migrate`
